5. **Open DevTools** to inspect elements

### Common Issues & Solutions
- **CSS not loading?** → Check Network tab, verify file exists
- **Dark background not showing?** → Check CSS variables in console
- **Layout broken on mobile?** → Check DevTools responsive mode
- **Chat not updating?** → Verify API running on 3001
- **Styles look wrong?** → Clear cache (Ctrl+Shift+Del), hard refresh
